# Theme Photos

Add 6 photos of Stan here, one for each theme. Each photo should match the theme's aesthetic.

## Required files:

1. **neon.png** - Electric, bold, vibrant vibe
2. **retro.png** - 80s synthwave, nostalgic feel
3. **minimalist.png** - Clean, elegant, simple photo
4. **dark.png** - Moody, mysterious, sleek
5. **sunset.png** - Warm, golden hour, cozy vibes
6. **cyberpunk.png** - Futuristic, tech-inspired

## Photo specs:
- Format: PNG
- Recommended size: 800x800 to 1200x1200 (square-ish)
- High quality, good lighting
- Each should visually match its theme's aesthetic
